## Asset Structure
Images should be placed in `/public/assets/images/` and categorized:
- `/sprites/`: Character and enemy sprites
- `/items/`: Item icons and chest images
- `/ui/`: Interface elements
- `/backgrounds/`: Map tiles or backdrop images

Example usage in code: `spriteUrl: "/assets/images/items/chest.png"`
